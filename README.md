# WanderWow

🚧 **Project Status: Under Active Development** 🚧

This project is currently being developed and improved. New features, UI enhancements, bug fixes, and deployment updates are being added regularly.

## Current Features
- User Authentication
- Create, Edit & Delete Listings
- Reviews & Ratings
- Session Management
- Flash Messages
- MongoDB Database Integration

## Planned Features
- Image Uploads with Cloudinary
- Search & Filtering
- Maps Integration
- Wishlist/Favorites
- Responsive Mobile Design
- Production Deployment
